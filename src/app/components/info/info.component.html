ABOUT

eLogger is an open source progressive web app (PWA). The app itself is hosted on GitHub. It can either be run in popular browsers or be exported to home screen of mobile devices via Safari (iOS) or Chrome (Android) browser apps.

There is no backend. Therefore, no data is exchanged. All data remains on the user's own device. This app has no commercial purposes. The use is private.
